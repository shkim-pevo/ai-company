# /retro

Data-driven team retrospective analyzing commit history, work patterns, and shipping velocity.

## Usage

- `/retro` — last 7 days (default)
- `/retro 24h` — last 24 hours
- `/retro 14d` — last 14 days
- `/retro 30d` — last 30 days
- `/retro compare` — current week vs prior week

Load the `skills/retro/SKILL.md` skill for the full retrospective methodology.
