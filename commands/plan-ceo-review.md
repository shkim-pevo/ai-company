# /plan-ceo-review

Founder mode planning review. Pressure-test product direction with taste, user empathy, and long time horizon.

Three modes:
- **SCOPE EXPANSION** — Build the cathedral. Find the 10x version.
- **HOLD SCOPE** — Bulletproof what exists. Minimum viable completeness.
- **SCOPE REDUCTION** — Strip to essence. What's the true MVP?
