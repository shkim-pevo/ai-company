# AI 챔피언 프로그램 (AI Champions Program)

**Version**: v1.0 (2026.03.13)
**규모**: 20개 부서 × 1명 = 20명 AI 챔피언

---

## AI 챔피언이란?

부서 내에서 AI 도구(스킬, 에이전트, 커맨드, MCP)의 **활용을 선도하고 팀원들의 스킬 향상을 촉진**하는 담당자.

> 추가 직책이 아닌 **역할 부여** — 기존 업무의 10~15%를 AI 챔피언 활동에 투자

---

## 선정 기준

### 필수 조건

| 기준 | 설명 |
|------|------|
| **스킬 숙련도** | 부서 스킬 번들 중 3개 이상 L3+ |
| **AI 도구 활용** | `continuous-learning-v2` L2+ 달성 |
| **커뮤니케이션** | 팀 내 교육/발표 경험 있음 |
| **자발성** | 본인 지원 또는 팀장 추천 |

### 우대 조건

- 크로스팀 협업 경험
- 새로운 스킬/커맨드 시도 이력
- 문서화 습관 (learned patterns 기록)

## AI 챔피언 역할과 책임

### 주간 활동 (주 2~3시간)

| 활동 | 시간 | 설명 |
|------|:---:|------|
| 스킬 트라이얼 | 1h | 새 스킬/커맨드 1개 시도 + 후기 작성 |
| 팀 내 공유 | 30m | 팀 슬랙/미팅에서 팁 1개 공유 |
| 질문 대응 | 30m | 팀원의 AI 도구 관련 질문 답변 |
| 학습 로그 | 30m | 본인 + 팀 학습 진척 기록 |

### 월간 활동

| 활동 | 설명 |
|------|------|
| **미니 워크숍** | 부서 내 30분 워크숍 1회 (스킬 시연 or 실습) |
| **챔피언 미팅** | 20명 전원 월 1회 모임 — 베스트 프랙티스 공유 |
| **스킬 갭 리포트** | 부서 자가진단 취합 → 리더십팀 보고 |
| **개선 제안** | 새 스킬/에이전트/커맨드 필요성 제안 |

### 분기별 활동

| 활동 | 설명 |
|------|------|
| **크로스팀 세션** | 타 부서 챔피언과 합동 워크숍 진행 |
| **스킬 번들 리뷰** | 부서 스킬 번들 적절성 재검토 |
| **숙련도 리포트** | 부서 전체 L1→L4 변화 추적 보고 |

## 챔피언 성과 측정

### KPI (분기별)

| 지표 | 목표 |
|------|------|
| 부서 평균 숙련도 상승 | +0.5 레벨/분기 |
| 부서 스킬 활용률 | 번들 스킬 중 80%+ 월 1회 이상 사용 |
| 워크숍 진행 | 3회+/분기 |
| 팀원 L3+ 달성률 | 60%+ |
| 크로스팀 세션 | 1회+/분기 |

### 인센티브

| 성과 | 보상 |
|------|------|
| KPI 달성 | **AI 챔피언 뱃지** + 리더십팀 표창 |
| 우수 챔피언 (상위 3명) | **다음 분기 스킬 번들 설계 참여** |
| 신규 스킬 기여 | **스킬 저자(Author)로 크레딧** |

## 챔피언 네트워크 운영

### 월간 챔피언 미팅 (20명)

```
시간: 매월 첫째 주 금요일 16:00 (1시간)
형식: 온라인/오프라인

아젠다:
1. [10분] 이번 달 하이라이트 — 가장 효과적이었던 스킬/사례
2. [20분] 챌린지 공유 — 팀에서 겪은 어려움 + 해결 방법
3. [20분] 크로스팀 매칭 — 다음 달 합동 세션 파트너 결정
4. [10분] 새 스킬/커맨드 소개 — 신규 추가된 도구 브리핑
```

### 슬랙 채널

```
#ai-champions — 챔피언 전용 채널
  - 새 스킬 시도 후기
  - 팁 & 트릭 공유
  - 질문/답변
  - 월간 KPI 공유
```
