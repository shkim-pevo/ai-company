# 스킬 자가진단 템플릿 (Skill Self-Assessment)

**Version**: v1.0 (2026.03.13)
**대상**: 전 부서 82명 담당자
**주기**: 분기별 1회 (Q1, Q2, Q3, Q4)
**소요시간**: 약 10~15분

---

## 진단 방법

### Step 1: 본인 부서의 스킬 번들 확인
`skill-bundles.md`에서 본인 부서 → 본인 역할의 **목표 레벨** 확인

### Step 2: 현재 레벨 자가 평가
아래 기준표를 참고하여 각 스킬에 대해 현재 레벨을 기입

### Step 3: 갭 산출
목표 레벨 - 현재 레벨 = 갭 (양수면 학습 필요, 0이면 달성, 음수면 초과 달성)

---

## 레벨 기준표

| 레벨 | 명칭 | 판단 기준 | 자가 체크 질문 |
|:---:|------|----------|--------------|
| **L1** | 인식 | 개념을 알고 있음 | "이 스킬이 뭔지 설명할 수 있는가?" |
| **L2** | 기초 | 가이드를 보며 수행 가능 | "문서를 참고하면 혼자 할 수 있는가?" |
| **L3** | 독립 | 도움 없이 혼자 수행 가능 | "가이드 없이도 실무에 바로 적용할 수 있는가?" |
| **L4** | 전문 | 남에게 가르칠 수 있음 | "다른 팀원에게 워크숍을 진행할 수 있는가?" |

## 제출 및 관리

1. 각 담당자가 자가진단 완료 후 **팀장에게 제출**
2. 팀장이 부서 종합 진단표 작성 → **리더십팀에 제출**
3. 리더십팀이 전사 스킬 맵 업데이트
4. 분기별 변화 추적 및 개선도 측정
