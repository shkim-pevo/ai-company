# 세일즈팀 (Sales Team)

## 👤 팀장 프로필
- **이름**: 세일즈팀장
- **역할**: 영업 및 매출 달성
- **보고 대상**: 리더십팀

## 🎯 팀 미션
고객에게 가치를 전달하고 매출 목표 달성

## 👥 팀 구성

| 직급 | 이름 | 역할 | 주요 책임 |
|------|------|------|---------|
| 팀장 | 세일즈팀장 | 영업 전략 수립 | 전략 기획, 팀 관리, 매출 보고 |
| 직원 | 파트너십 매니저 | 파트너 관리 | 크레소티/대교 관계, 신규 파트너 발굴 |
| 직원 | 영업 담당 | 고객 온보딩 | 약국 온보딩, 파일럿 관리, 계약 체결 |

## 🔄 업무 프로세스

### 단계별 워크플로우: 팀장 배정 → 담당자 실무 → 팀장 검토/검증 → 보고

#### 1. 팀장 배정 (세일즈팀장)
```
FROM: 리더십팀
TO: 세일즈팀장

영업 목표:
- 분기 목표: [$금액]
- 타겟 고객: [약국 체인/개별 약국]
- 제품: [제품명]
- 기한: [분기]
- 담당자: [배정할 팀원]

팀장의 역할:
→ 목표 명확화
→ 담당자 선정 및 자원 배정
→ 파이프라인 수립
→ 진행 상황 모니터링
```

#### 2. 담당자 실무 (파트너십/영업/CRM)
```
담당자의 역할:
→ 상세 계획 수립
→ 고객/파트너 접촉
→ 미팅, 데모, 제안 진행
→ 고객 데이터 기록
→ 중간 진행 상황 팀장 보고

예시:
- 파트너십 매니저: 파트너사 미팅, 협상, 계약 추진
- 영업 담당: 약국 미팅, 제안서 작성, 계약 협상
- CRM 담당: 거래 기록, 파이프라인 업데이트, 분석
```

#### 3. 팀장 검토/검증 (세일즈팀장)
```
FROM: 담당자
TO: 세일즈팀장

검토 항목:
□ 영업 진행 상황
□ 거래 규모 및 조건
□ 고객 적격성
□ 계약서 검토

팀장의 역할:
→ 거래 진행 상황 확인
→ 계약 조건 검토
→ 클로징 전략 수립
→ 최종 승인
```

#### 4. 보고 (세일즈팀장)
```
FROM: 세일즈팀장
TO: 리더십팀

영업 현황 리포트:

## 성과 지표
- 파이프라인: [$총액]
- 계약 체결: [건수/$금액]
- 목표 달성률: [%]
- 평균 계약 금액: [$]
- 계약 주기: [일수]

## 주요 고객 및 거래
[진행 중인 주요 거래]

## 다음 주 계획
[예정된 미팅 및 액션]
```

## 🔗 협업 팀
- **← 마케팅팀**: 리드 수령
- **↔ 개발팀**: 제품 정보 및 데모 지원
- **↔ 고객지원팀**: 고객 정보 공유 및 온보딩 지원
- **→ 리더십팀**: 매출 보고 및 성과 분석
