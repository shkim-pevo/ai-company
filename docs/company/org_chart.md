## 5단계 보고 체계

```
접수 → 실무 → 취합 → 경영진 → 최종보고
 ①      ②      ③       ④        ⑤
```

| 단계 | 내용 | 담당 |
|------|------|------|
| ① 접수 | CEO 지시 또는 팀 자체 발의 | 리더십팀 → 해당 팀 |
| ② 실무 | 각 팀 병렬 분석/작업 수행 | 담당 팀(들) |
| ③ 취합 | 팀 간 산출물 통합 + 크로스체크 | 종합팀 |
| ④ 경영진 | 리뷰 + 승인/반려 | 리더십팀 |
| ⑤ 최종보고 | CEO에게 최종 산출물 전달 | 종합팀 → CEO |
