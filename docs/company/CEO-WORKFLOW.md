# 👔 CEO 업무 지시 체계

**CEO: 김상현**

## 📊 조직 구조
```
CEO 김상현
    ↓
리더십팀 (CTO, CPO, CMO, CFO 등)
    ↓
각 팀 팀장 (17개 팀)
    ↓
팀원 (AI Agents)
```

## 🔄 업무 흐름

### 1단계: CEO → 리더십팀
```markdown
# 업무 지시서
작성자: CEO
일시: YYYY-MM-DD
우선순위: [긴급/높음/보통/낮음]

## 업무 내용
[구체적인 업무 설명]

## 목표
[달성해야 할 목표]

## 기한
[완료 기한]

## 관련 팀
[예상되는 관련 팀들]
```

### 2단계: 리더십팀 → 팀장들
리더십팀이 업무를 분석하고 적절한 팀에 배분

### 3단계: 팀장 → 팀원
팀장이 구체적인 작업으로 분해하여 팀원에게 할당

### 4단계: 팀원 → 팀장
팀원들이 작업 완료 후 팀장에게 보고

### 5단계: 팀장 → 리더십팀
팀장이 결과를 취합하여 리더십팀에 보고

### 6단계: 리더십팀 → CEO
리더십팀이 검토 후 최종 의견과 함께 CEO에게 보고

## 📝 사용 예시

### CEO의 업무 지시
```
리더십팀에게:

새로운 AI 챗봇 제품 개발을 시작하려고 합니다.

목표:
- 시장 분석 및 경쟁사 조사
- 제품 컨셉 및 MVP 계획 수립
- 3개월 내 베타 버전 출시

기한: 2주 내 초기 보고서
우선순위: 높음

적절한 팀들에게 업무를 배분하고,
각 팀의 결과를 종합하여 보고해주세요.
```

### 리더십팀의 업무 분배
```
[리더십팀 → 조사팀]
시장 분석 및 경쟁사 조사 요청

[리더십팀 → 기획팀]
제품 컨셉 및 MVP 계획 수립 요청

[리더십팀 → 재무팀]
예산 분석 및 ROI 예측 요청
```

## 🗂️ 파일 저장 위치

- CEO 지시사항: `leadership/tasks/ceo-requests/`
- 팀별 업무: `[팀명]/tasks/`
- 팀장 보고서: `[팀명]/outputs/reports/`
- 최종 보고: `leadership/outputs/final-reports/`

## ⚡ 빠른 명령어

### 업무 지시
```bash
# CEO 업무 폴더로 이동
cd ~/ai-company/leadership/tasks/ceo-requests

# 새 업무 지시서 생성
# 파일명: YYYYMMDD-업무명.md
```

### 진행 상황 확인
```bash
# 모든 팀의 진행 중인 업무 확인
find ~/ai-company/*/tasks -type f -name "*.md"

# 완료된 산출물 확인
find ~/ai-company/*/outputs -type f -name "*.md"
```
