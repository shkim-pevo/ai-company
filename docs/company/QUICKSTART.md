# 🚀 빠른 시작 가이드

## Claude Code Agent Teams로 회사 운영하기

### 1️⃣ 첫 프로젝트 시작

```bash
cd ~/ai-company/projects
mkdir my-first-project
cd my-first-project
```

### 2️⃣ Claude Code 실행

```bash
claude
```

### 3️⃣ Agent Team 생성 예시

#### 예시 1: 새로운 제품 아이디어 검증
```
조사팀, 토론팀, 종합팀으로 구성된 팀을 만들어줘.

작업:
1. 조사팀: AI 챗봇 시장 조사 (경쟁사, 시장 규모, 트렌드)
2. 토론팀: 차별화 포인트 토론
3. 종합팀: 최종 인사이트 정리

각 팀은 ~/ai-company/[팀명]/outputs/ 에 결과를 저장해줘.
```

#### 예시 2: 제품 개발 프로젝트
```
기획팀, 디자인팀, 개발팀, 퀄리티팀으로 팀을 구성해줘.

프로젝트: 할 일 관리 웹 앱 개발

워크플로우:
1. 기획팀: PRD 작성 및 사용자 스토리 정의
2. 디자인팀: UI/UX 설계 및 프로토타입
3. 개발팀: 프론트엔드 및 백엔드 구현
4. 퀄리티팀: 테스트 케이스 작성 및 검증

각 단계별 산출물을 해당 팀 디렉토리에 저장해줘.
```

#### 예시 3: 전사 전략 수립
```
리더십팀을 중심으로 조사팀, 데이터 분석팀, 재무팀이 협업하는 팀을 만들어줘.

목표: 2026 Q2 OKR 수립

1. 조사팀: 시장 기회 분석
2. 데이터 분석팀: 현재 성과 지표 리뷰
3. 재무팀: 예산 및 ROI 분석
4. 리더십팀: OKR 초안 작성 및 우선순위 결정

결과를 ~/ai-company/leadership/okrs/2026-Q2.md에 저장해줘.
```

### 4️⃣ 팀별 프롬프트 활용

각 팀의 `prompts/` 디렉토리에 자주 쓰는 프롬프트를 저장하세요:

```bash
# 예시: 조사팀 프롬프트
echo "시장 조사를 위한 체크리스트:
1. 시장 규모 (TAM, SAM, SOM)
2. 주요 경쟁사 3-5개
3. 가격 정책
4. 핵심 기능
5. 고객 리뷰 분석" > ~/ai-company/research/prompts/market-research.md
```

### 5️⃣ 협업 워크플로우

```
리더십팀에게: 
다음 주제로 전체 팀 미팅을 진행해줘.

주제: "새로운 AI 제품 론칭 전략"

참여 팀: 조사팀, 기획팀, 마케팅팀, 재무팀
각 팀은 자신의 관점에서 의견을 제시하고,
리더십팀은 최종 결정과 액션 아이템을 정리해줘.

결과는 ~/ai-company/leadership/decisions/ 에 저장해줘.
```

