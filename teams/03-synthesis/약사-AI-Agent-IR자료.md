# 1. Executive Summary

## "약사를 대체하지 않습니다. 약사를 슈퍼 약사로 만듭니다."

PharmAI는 **한국 최초의 약사 전용 AI Agent 플랫폼**입니다. RAG(Retrieval-Augmented Generation) 기반 아키텍처와 Co-pilot 모드를 통해, 약사의 조제 오류를 줄이고, 복약지도를 자동화하며, 약물 상호작용을 실시간 검사하는 지능형 솔루션을 제공합니다.

### 핵심 수치 요약

| 구분 | 수치 |
|------|------|
| **글로벌 약국 시장** | $1.35조 (2024) → $2.06조 (2030), CAGR 7.3% |
| **약국 자동화 시장** | $71.9억 (2025) → $166.5억 (2034), CAGR 10.12% |
| **헬스케어 AI Agent 시장** | CAGR 44.1%, 2030년 $69.2억 |
| **한국 SAM** | 1,500~2,000억 원 |
| **3년차 SOM 목표** | 640억 원 (기본 시나리오) |
| **투자 요청** | 시드 라운드 10~20억 원 |

### 왜 지금인가?

- 2025년 한국 **초고령사회 진입** (65세 이상 20.6%) -- 다제복용 환자 급증
- 식약처 **세계 최초** 생성형 AI 의료기기 가이드라인 발표 (2025.1)
- **Fast-Track** 인허가 경로 신설 (2026.1) -- 규제 부담 최소화
- 한국에 **약사 AI Agent 솔루션 전무** -- 선점 기회

### 토론팀 결론: 조건부 추진(Conditional Go)

린(Lean) 투자로 MVP를 빠르게 구축하고, 파일럿을 통해 제품-시장 적합성(PMF)을 검증한 후 단계적으로 확장합니다.

# 3. 솔루션

## 3.1 제품 개요: PharmAI Agent

**"약사의 두뇌를 확장하는 AI Co-pilot"**

PharmAI는 약사가 더 정확하고, 더 빠르게, 더 안전하게 업무를 수행할 수 있도록 돕는 AI Agent 플랫폼입니다.

### 핵심 운영 원칙: Co-pilot 모드

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   처방전 입력  │────→│  AI 분석/제안  │────→│  약사 최종 확인│
│  (자동 OCR)   │     │  (DDI, 용량등) │     │  (승인/수정)  │
└──────────────┘     └──────────────┘     └──────────────┘
                           │
                     AI가 제안하고
                     약사가 결정한다
```

- **AI는 제안(Suggest)** -- 약물 상호작용 경고, 복약지도 초안, 용량 검증
- **약사가 결정(Decide)** -- 최종 판단은 항상 약사의 임상적 전문성에 의존
- **인허가 부담 최소화** -- 정보 제공형 Co-pilot으로 의료기기 규제 범위 최소화

## 3.2 핵심 기능

| 기능 | 설명 | 기술 수준 |
|------|------|-----------|
| **DDI 검사** | 약물-약물 상호작용 실시간 검사 | 정확도 92%+ |
| **AI 복약지도** | 환자 맞춤형 복약지도문 자동 생성 (연령/질환/다국어) | RAG 기반 |
| **처방전 분석** | OCR 기반 처방전 자동 입력 + 이상 패턴 탐지 | 정확도 96% |
| **용량 검증** | 연령/체중/신기능 기반 용량 적절성 자동 검증 | 규칙+AI 하이브리드 |
| **알림 지능화** | 기존 DUR 알림의 우선순위화 및 맥락 기반 필터링 | ML 기반 |
| **보험 청구 보조** | 심사 반려 패턴 분석 및 사전 검증 | 데이터 기반 |

## 3.3 기술 차별점

### RAG 기반 아키텍처

```
┌───────────────────────────────────────────────────┐
│                  PharmAI 아키텍처                    │
│                                                     │
│  ┌─────────┐   ┌──────────────┐   ┌─────────────┐ │
│  │ 약물 DB  │──→│  RAG Engine   │──→│ LLM 추론    │ │
│  │(TRL 7-8)│   │(DrugRAG 적용)│   │(환각 최소화)│ │
│  └─────────┘   └──────────────┘   └─────────────┘ │
│       │              │                    │         │
│  건보 DB 연동    약학 지식 검색        Co-pilot 출력  │
│  DUR DB 연동    근거 기반 응답          약사 확인     │
│  약학정보원 DB   출처 표시 의무화                      │
└───────────────────────────────────────────────────┘
```

| 기술 요소 | 효과 |
|-----------|------|
| **RAG (Retrieval-Augmented Generation)** | 환각(Hallucination) 최소화, 근거 기반 응답 |
| **DrugRAG 적용** | 약물 정보 정확도 **7~21%p 향상** |
| **약사+AI CDSS 결합** | 약사 단독 대비 의사결정 정확도 **32% 향상** |
| **약물 DB 연동 (TRL 7-8)** | 기술 성숙도 검증 완료, 상용화 준비 단계 |
| **Co-pilot 모드** | 인허가 부담 최소화 + 약사 수용성 극대화 |

**핵심 메시지:** PharmAI는 "정보 제공형 Co-pilot"으로 포지셔닝하여 의료기기 인허가 부담을 최소화하면서도, 약사의 의사결정 품질을 32% 향상시키는 실질적 가치를 제공합니다.

| **약국 현장 경험** | 약사 자문단 운영, 현장 피드백 기반 개발 | 제품-시장 적합성 |

## 6.4 잠재 위협과 대응

| 위협 | 위협도 | 대응 전략 |
|------|--------|----------|
| **네이버 HyperClova Med** | 중-높음 | 약국 현장 특화 vs 범용 의료 AI, 선점 후 데이터 해자 구축 |
| **유비케어 AI 확장** | 중간 | 속도 우위(AI Native vs 레거시 전환), 기술 파트너십 제안 |
| **글로벌 기업 한국 진출** | 중-낮음 | 한국어/규제 장벽, 로컬 파트너십 선점 |

**핵심 메시지:** 한국에는 약사 AI Agent가 전무합니다. 글로벌 경쟁사(Asepha, Foundation Health)는 아직 한국에 진출하지 않았고, 국내 기존 솔루션은 AI Agent 수준에 미달합니다. **지금이 선점의 골든 타임입니다.**
